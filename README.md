# Optimizing Retail Performance: Sales Growth Prediction  
### *Predictive Analytics & Machine Learning with Microsoft Azure*  
**Capstone Project 3**

This project analyzes Zara retail sales data to identify key sales drivers and build predictive models using Microsoft Azure Machine Learning. The goal is to understand how pricing, placement, promotions, and product categories influence sales performance and use these insights for forecasting and decision-making.

---

## Introduction & Business Problem

Prior analysis identified key drivers of sales, including the impact of product placement, pricing strategies, and promotional activities on transaction volume and revenue.

The findings revealed that:

- High-traffic product placements increase sales volume  
- Mid-range pricing strategies generate the highest revenue  
- Promotions boost customer engagement without significantly changing spending behavior  

Building on these findings, this project shifts toward predictive modeling using Microsoft Azure to forecast sales growth and support data-driven retail strategies.

The objective is to transform historical retail insights into actionable predictions that improve:

- Merchandising strategy  
- Inventory planning  
- Pricing optimization  
- Promotional effectiveness  
- Overall retail performance  

---

## Key Modeling Questions To Solve

- Can Microsoft Azure uncover patterns in Zara’s sales data for better store performance?  
- How accurately can sales growth be predicted using historical retail data?  
- Which variables are the most significant predictors of sales performance?  
- Can machine learning models classify high & low-performing products or store placements?  
- Are there distinct clusters of products or stores with similar sales behaviors?  
- How can predictive insights inform pricing, promotion, and merchandising strategies?
